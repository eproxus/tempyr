<h1 align="center">Tempyr</h1>

<p align="center">A mod manager for Hytale, built with Avalonia on .NET 10.</p>

Tempyr lets you view, manage, and update your installed Hytale mods. It auto-detects your Hytale installation, reads mod manifests from your mods folder, and checks CurseForge for available updates.

**Features**
- Auto-detects Hytale install path via registry
- Lists installed mods with name, version, author, and description
- Checks for and installs updates via CurseForge for mods with a known source
- Enable/disable and remove mods
